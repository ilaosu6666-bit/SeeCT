# 演示视频录制指南

## 要求

- **时长**：不超过 4 分钟
- **格式**：MP4（推荐 1920×1080, 30fps）
- **内容**：完整展示作品核心功能

## 推荐录制工具

- **Windows**：Xbox Game Bar（Win + G，免费）/ OBS Studio（免费）
- **macOS**：QuickTime Player（文件 → 新建屏幕录制）
- **通用**：OBS Studio（https://obsproject.com/）

## 分镜脚本（建议4分钟）

### 0:00-0:30 — 作品介绍
- 展示首页（https://zhiyingsuyuan.streamlit.app）
- 简单说明："智影溯源是一个AI教学平台，让公众以AI研究员视角体验训练肺结节诊断AI"
- 侧边栏切换模块

### 0:30-1:15 — 关卡1 数据猎人（45秒）
- 点击进入AI训练师
- Tab1：快速浏览CT知识卡片和3D切片浏览器（拖动滑块几次）
- Tab2：展示肺窗调节实验，调节窗宽窗位 → 展示肺窗效果
- Tab3：浏览2-3张已标注样本
- 进度追踪：三个勾全绿 → 进入下一关

### 1:15-1:45 — 关卡2 架构师（30秒）
- 选择ResNet18架构
-快速浏览超参数配置面板
- 点击确认并进入训练

### 1:45-2:45 — 关卡3 训练大师（60秒）⭐核心
- 展示训练启动阶段 → 曲线图
- 快速推进到决策点1、2、3，展示不同的情景说明
- 训练完成后展示完整曲线
- **重点：拖动Grad-CAM演变滑块**，展示从epoch 1（多个随机热点）→ epoch 50（精准聚焦）的变化
- 这是作品的**核心亮点画面**

### 2:45-3:15 — 关卡4 诊断专家（30秒）
- 选择一个病例，进行独立诊断判断
- 点击启动AI分析 → 展示Grad-CAM热力图叠加
- 展示诊断评分

### 3:15-3:45 — 关卡5 临床实战（30秒）
- 快速浏览批量筛查界面
- 完成几张筛查 → 展示最终结算面板
- 展示综合能力评估雷达图和总评级

### 3:45-4:00 — 总结（15秒）
- 展示成就列表
- 结束语："智影溯源——让每个人都能理解AI如何看病"

## 录制提示

1. **关闭通知**：录制前关闭所有桌面通知
2. **鼠标操作清晰**：点击前停顿半秒，让观众看清楚操作
3. **语速适中**：控制在每分钟 150-180 字
4. **Dev模式跳关**：为快速展示，可用侧边栏开发者模式直接跳转到各关卡
5. **先练一遍**：完整走一次流程，确保所有内容正常加载
