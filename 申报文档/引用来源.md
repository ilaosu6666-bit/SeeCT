# 引用来源

## 数据集

### LIDC-IDRI (Lung Image Database Consortium - Image Database Resource Initiative)

- **全称**：The Lung Image Database Consortium (LIDC) and Image Database Resource Initiative (IDRI)
- **描述**：包含 1018 个胸部CT病例的公开数据集，每个病例由4位放射科专家标注了肺结节的位置和9项影像学特征
- **来源**：https://wiki.cancerimagingarchive.net/display/Public/LIDC-IDRI
- **引用**：Armato SG 3rd, et al. "The Lung Image Database Consortium (LIDC) and Image Database Resource Initiative (IDRI): A Completed Reference Database of Lung Nodules on CT Scans." Medical Physics, 2011; 38(2):915-931.
- **使用方式**：本项目使用LIDC-IDRI中的4个病灶患者和2个正常患者的CT切片构建教学病例库。原始DICOM文件已转换为PNG格式用于Web展示。

### LUNA16 (LUng Nodule Analysis 2016)

- **描述**：基于LIDC-IDRI的子集，包含888个CT扫描，专门用于肺结节检测算法评估
- **来源**：https://luna16.grand-challenge.org/
- **引用方式**：本项目的数据预处理管线参考了LUNA16的数据格式和处理流程

## 深度学习模型

### ResNet (Residual Network)

- **论文**：He K, Zhang X, Ren S, Sun J. "Deep Residual Learning for Image Recognition." CVPR 2016.
- **DOI**：https://doi.org/10.1109/CVPR.2016.90
- **使用方式**：本项目使用PyTorch官方实现的ResNet18作为分类骨干网络，将最后一层全连接层替换为2分类输出（有结节/无结节）

### PyTorch

- **框架**：https://pytorch.org/
- **许可**：BSD License
- **引用**：Paszke A, et al. "PyTorch: An Imperative Style, High-Performance Deep Learning Library." NeurIPS 2019.

## 可解释AI方法

### Grad-CAM (Gradient-weighted Class Activation Mapping)

- **论文**：Selvaraju RR, Cogswell M, Das A, Vedantam R, Parikh D, Batra D. "Grad-CAM: Visual Explanations from Deep Networks via Gradient-based Localization." ICCV 2017.
- **DOI**：https://doi.org/10.1109/ICCV.2017.74
- **使用方式**：本项目实现了Grad-CAM的PyTorch版本，通过hook机制提取ResNet18最后一层卷积特征图，计算类别梯度权重，生成7×7热力图后上采样叠加到原始CT图像上。

### Guided Backpropagation

- **论文**：Springenberg JT, Dosovitskiy A, Brox T, Riedmiller M. "Striving for Simplicity: The All Convolutional Net." ICLR 2015 Workshop.
- **使用方式**：本项目实现了Guided Backpropagation的PyTorch版本，通过修改ReLU层的反向传播hook只传递正值梯度，展示模型关注的细粒度纹理特征。
