# 科学原理及内容

## 一、CT影像基础

### 1.1 CT成像原理

CT（Computed Tomography，计算机断层扫描）利用X射线从多个角度穿透人体，经计算机重建得到人体横断面图像。CT值以HU（Hounsfield Unit）为单位，表示组织对X射线的吸收系数：

| 组织 | CT值 (HU) |
|------|-----------|
| 空气 | ≈ -1000 |
| 肺组织 | ≈ -700 ~ -800 |
| 水 | 0 |
| 软组织（结节） | +20 ~ +50 |
| 骨骼 | +400 ~ +1000 |

正是肺组织（-800HU）与软组织结节（+30HU）之间约830HU的密度差异，使得CT能够清晰显示肺内结节。这也是AI识别肺结节的基础——模型学习到的其实是HU值在空间上的分布模式。

### 1.2 肺窗技术

原始CT的HU值范围极宽（-1000~+1000），人眼只能分辨约16个灰阶。**窗技术**通过设定窗宽(WW)和窗位(WC)来选择显示特定HU范围：

- **肺窗（WC=-600, WW=1500）**：显示-1350~+150HU范围，突出肺实质和结节
- **纵隔窗（WC=50, WW=350）**：显示-125~+225HU范围，观察纵隔结构
- **骨窗（WC=400, WW=1800）**：显示-500~+1300HU范围，观察骨骼细节

本作品的"肺窗调节实验"让用户亲手调整窗宽窗位，理解这一临床基础操作。

## 二、深度学习原理

### 2.1 卷积神经网络（CNN）

CNN通过层级结构从图像中提取特征：

1. **浅层**（前面几层卷积）：识别边缘、纹理、颜色等低级特征
2. **中层**：组合低级特征，识别形状、轮廓
3. **深层**：组合中层特征，识别完整物体（如肺结节）

本作品采用 **ResNet18** 架构，核心创新是**残差连接（Residual Connection）**——每层不仅学习新特征，还保留上一层的原始信息。这解决了深层网络"梯度消失"的问题，让18层网络也能稳定训练。

### 2.2 训练过程

神经网络通过**反向传播**学习：

1. **前向传播**：输入CT图像 → 逐层计算 → 输出预测（有结节/无结节）
2. **计算损失**：将预测结果与真实标签对比，量化误差（CrossEntropyLoss）
3. **反向传播**：从输出层逐层往回计算梯度，确定每个参数对误差的贡献
4. **梯度下降**：沿梯度反方向调整参数，使下一次预测更准确

这个循环在全部训练数据上重复多轮（Epoch），模型性能逐步提升。本作品的关卡3将这个抽象过程可视化为曲线和热力图。

### 2.3 过拟合与泛化

- **欠拟合**：模型太简单，训练集和验证集表现都差
- **正常拟合**：模型学会了通用规律，训练集和验证集表现接近
- **过拟合**：模型"背答案"——记住了训练数据的细节但不会应对新数据（训练准确率高但验证准确率低）

本作品的关卡3在Epoch 25故意展示过拟合现象，让用户选择应对策略，理解正则化的重要性。

## 三、可解释AI（XAI）原理

### 3.1 为什么需要可解释AI

传统深度学习模型是"黑箱"——给出输入，得到输出，但不知道模型为什么做出这个判断。在医学领域，这尤其危险：
- AI可能因为图像中的医院标记而非病灶本身做出判断
- AI可能关注了无关区域（如肺纹理、肋骨边缘）

### 3.2 Grad-CAM（梯度加权类激活映射）

**Grad-CAM** 是当前最广泛使用的CNN可视化方法，原理如下：

1. 输入一张CT图像，模型做出预测
2. 计算预测类别相对于最后一层卷积特征图的**梯度**
3. 对梯度做全局平均池化，得到每个特征图通道的**权重**（代表该通道对最终预测的贡献）
4. 将权重与特征图加权求和，得到**类激活热力图**
5. 热力图叠加到原图上，红色区域 = AI最关注的区域

公式简化表述：**CAM = ReLU(Σ(权重ᵢ × 特征图ᵢ))**

### 3.3 Guided Backpropagation

与Grad-CAM互补：
- **Grad-CAM** 回答"哪里重要"（粗粒度定位）
- **Guided Backprop** 回答"什么样的纹理/边缘重要"（细粒度特征）

两者结合，全面解释AI的判断依据。
