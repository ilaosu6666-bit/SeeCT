# 作品简介

## 基本信息

| 项目 | 内容 |
|------|------|
| **作品名称** | 智影溯源 — 基于可解释AI的肺结节交互式教学平台 |
| **英文名称** | SeeCT: Explainable AI-based Interactive Teaching Platform for Lung Nodules |
| **作品类型** | H5 科普互动作品（互动科普小游戏 + 虚拟科普实验） |
| **在线地址** | https://zhiyingsuyuan.streamlit.app |

## 一句话概述

**让公众以"AI研究员"的视角，从零开发一个肺结节诊断AI，在游戏中理解深度学习与可解释AI的原理。**

## 核心功能

本作品是一个**五关游戏化教学系统**，玩家扮演AI研究员，完整体验AI模型的开发全流程：

1. **关卡1 — 数据猎人**：浏览3D CT切片、调节肺窗参数、理解医学影像数据
2. **关卡2 — 架构师**：选择神经网络架构（ResNet18 / 3D CNN）、配置训练超参数
3. **关卡3 — 训练大师（核心）**：实时观察AI学习曲线、在关键决策点做出选择（应对过拟合等）、观看Grad-CAM注意力演变
4. **关卡4 — 诊断专家**：AI辅助诊断真实CT病例、用户圈选区域与AI热力图对比（IoU评分）
5. **关卡5 — 临床实战**：模拟放射科批量筛查、识别AI可能的误判

另有独立的 **CT分析演示** 模块，支持上传CT图像进行实时AI分析。

## 创新点

1. **沉浸式游戏化教学**：不是被动看演示或答题，而是以"做决策→看反馈→理解原理"的循环完成学习
2. **可解释AI可视化**：Grad-CAM热力图逐轮演变，直观展示AI从"到处乱看"到"精准定位病灶"的学习过程
3. **真实医学数据**：使用 LIDC-IDRI 公共数据集的真实CT切片，病例来自4个病灶患者和2个正常对照
4. **人机协作诊断**：用户标注区域与AI关注区域IoU对比，展示AI辅助临床诊断的实际应用场景

## 技术亮点

- **深度学习**：PyTorch + ResNet18 实现肺结节二分类，训练准确率 >82%
- **可解释AI**：Grad-CAM + Guided Backpropagation 双重视角解释AI决策
- **在线部署**：Streamlit Cloud 一键访问，无需安装任何软件
- **离线可用**：提供完整源码和一键启动脚本，支持本地运行
