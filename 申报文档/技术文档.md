## 核心模块说明

### game.py — 游戏引擎

五关游戏系统的核心实现，约1800行代码。

主要组件：
- `init_game_state()` — 游戏状态初始化
- `render_sidebar()` — 侧边栏（进度、分数、成就、开发者模式）
- `render_stage1()` ~ `render_stage5()` — 五个关卡渲染
- `_generate_fake_cam()` — 模拟Grad-CAM热力图（4阶段演变）
- `_plot_training_curves()` — 训练曲线绘制（截断式显示）
- `_show_gradcam_snapshot()` / `_show_gradcam_evolution()` — 热力图展示
- `load_model_resource()` — 模型缓存加载

### part1.py — CT分析工具

独立的CT分析模块，包含：
- `GradCAMHelper` — Grad-CAM生成（PyTorch hook实现）
- `GuidedBackpropHelper` — Guided Backpropagation生成
- `build_model()` — ResNet18模型构建
- `_load_image_smart()` — 智能图片加载（支持PNG/NPY/16bit）
- CT分析页面：病例库浏览、Grad-CAM热力叠加、用户圈选IoU对比

### model.py — 训练脚本

模型训练管线：
- 数据加载（CTDataset）
- 数据增强（随机水平翻转、旋转）
- ResNet18训练（Adam优化器，CrossEntropyLoss）
- 训练历史持久化（JSON）
- 每轮Grad-CAM快照保存
- 模型checkpoint保存

## 部署方式

### 在线部署（已运行）

**平台**：Streamlit Cloud
**地址**：https://zhiyingsuyuan.streamlit.app
**方式**：GitHub Push 自动触发部署

### 离线运行

**前置要求**：Python 3.10+ 和 pip

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
streamlit run app.py --server.port 8501

# 3. 浏览器打开
# http://localhost:8501
```

或使用 offline/ 目录中的一键启动脚本。

## 训练模型

如需重新训练模型（生成 training_history.json 和 Grad-CAM 快照）：

```bash
python model.py
```

训练数据：cases/Lesion/（病灶CT）+ cases/Normal/（正常CT）
输出：model_parameter/*.pt + loss_fig/*.json + loss_fig/gradcam_epochs/*.npy
