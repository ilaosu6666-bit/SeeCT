# 智影溯源 — 离线运行说明

## 环境要求

- **操作系统**：Windows 10+ / macOS 11+ / Linux (Ubuntu 20.04+)
- **Python**：3.10 或更高版本
- **浏览器**：Chrome 90+ / Safari 14+ / Firefox 88+

## 快速启动

### Windows

双击 `start.bat`

### macOS / Linux

```bash
chmod +x start.sh
./start.sh
```

## 手动启动

```bash
# 1. 创建虚拟环境（首次）
python -m venv .venv

# 2. 激活虚拟环境
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动
streamlit run app.py --server.port 8501

# 5. 浏览器打开
# http://localhost:8501
```

## 注意事项

1. 首次启动需要下载PyTorch等依赖（约 800MB），请确保网络畅通
2. 在线部署版本无需安装任何软件：https://zhiyingsuyuan.streamlit.app
3. 如需训练模型：`python model.py`（需要GPU推荐，CPU也可）
